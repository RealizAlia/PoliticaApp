Política de Segurança

Esta Política de Segurança descreve as medidas adotadas pelo nosso aplicativo de moeda social para garantir a proteção e privacidade dos dados dos usuários.

Coleta e Uso de Dados

Coletamos e armazenamos apenas os dados necessários para a operação do aplicativo, incluindo nome, CPF, data de nascimento, e-mail e telefone. Todas as informações são utilizadas estritamente para os propósitos do serviço.

Proteção de Dados

Utilizamos criptografia de ponta a ponta para proteger as informações dos usuários, além de práticas avançadas de segurança, como autenticação multifator (MFA) e monitoramento contínuo contra atividades suspeitas.

Compartilhamento de Informações

Os dados dos usuários não são compartilhados com terceiros sem consentimento explícito, exceto quando exigido por lei ou para a execução de serviços essenciais à plataforma.

Direitos dos Usuários

Os usuários podem solicitar a exclusão ou atualização de seus dados a qualquer momento, conforme previsto pela Lei Geral de Proteção de Dados (LGPD).

Contato

Para dúvidas ou solicitações, envie um e-mail para contato@redealia.com
